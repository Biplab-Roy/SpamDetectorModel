A simple spam message detector using NLP and Machine Learning. Now days most of us use Gmail, where this technology is being used widely. There are many more application for spam message detector. Natural language processing or NLP is currently one of the major successful application areas for deep learning. The model is embeded in contact form. You can fill up the contact form with dummy information. Hit submit to see the result. Model has accuracy of 96.5% on test data.

Technology


NLP & ML

  • nltk (Text Processing)
  • Naive Bayes (ML algorithm)
  • Some more external library like pandas, numpy, sklearn

Web Technology

  • Flask

Others

  • Pickle (Connecting and Storing model)
